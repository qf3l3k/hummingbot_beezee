Initial Beezee DEX spot connector draft for Hummingbot.

Included:
- Beezee spot connector structure and exchange modules
- Market discovery and trading pair mapping
- Trading rule synthesis from chain metadata
- Order book snapshots and public trade polling
- Bank balances and Beezee dust balances
- Limit order create/cancel flow through Cosmos transaction signing
- Polling-based order status tracking and fill attribution
- Discovery notes, README, and mocked unit tests

Notes:
- Early release intended as the first functional connector baseline
- Real-world Beezee testnet/mainnet validation was still pending at this stage
