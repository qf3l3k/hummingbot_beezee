Added Beezee wallet usability improvements and testnet configuration support.

Included:
- Mnemonic wallet mode with BIP39 seed support
- Optional mnemonic passphrase support
- Configurable HD derivation path support
- Beezee testnet network mode
- Example connector config under `config/`
- Updated Beezee docs for testnet and mnemonic usage
- Extended unit tests for mnemonic and testnet configuration

Notes:
- This release improved setup flexibility but did not yet include the later hardening needed for reliable testnet trading
